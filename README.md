Gedeonix UI
-----------

Gedeonix UI is a simple set of controls based on css using flexbox and responsive.
This is a very simple project in alpha state :)

Demo
----

View html+css Gedeonix UI [demo](http://gedeonix.github.io/gedeonix-ui).

Install
-------

    npm install

Build
-----

    npm run dev
